# Urenregistratie

## 👤 Auteur

Umar

## 📖 Beschrijving

Dit programma vraagt gegevens via de command prompt en slaat deze op in een CSV-bestand.

---
### Problemen oplossen

Als het programma niet start, controleer dan of je in de juiste map zit.  
Gebruik het commando cd om naar de map te gaan waar het spel staat.  

Controleer vervolgens met ls of het bestand aanwezig is.  

Als het bestand zichtbaar is, kun je het programma starten met:


## ▶️ Hoe starten

```bash
node urenregistratie.js
```

---

## 🛠️ Gebruikte technieken

* JavaScript (Node.js)
* readline (voor input)
* fs (voor bestanden)

---

## 📄 Output

De gegevens worden opgeslagen in:
urenregistratie.csv
